# ヘッドレスCMS (microCMS) スキーマ定義設計書

運用フェーズ（Phase 4以降）で導入予定のmicroCMSスキーマ定義です。

## 1. ニュース（お知らせ）
API名: `news`
APIの型: リスト形式

| フィールド名 | フィールドID | 種類 | 必須 | 詳細設定・備考 |
| :--- | :--- | :--- | :--- | :--- |
| タイトル | `title` | テキストフィールド | 必須 | 文字数制限: 100文字程度 |
| スラッグ | `slug` | テキストフィールド | 必須 | URLの一部となる英語/数字の文字列（例: `website-renewal-2026`） |
| 日付 | `date` | 日時（Date） | 必須 | 公開日または記事の日付 |
| カテゴリ | `category` | セレクトボックス または 複数選択 | 必須 | 選択肢: 「お知らせ」「採用」「事業」「その他」 |
| ステータス | `status` | セレクトボックス | 必須 | 選択肢: `published`（公開）, `hidden`（非公開） |
| アイキャッチ画像 | `image` | 画像 | 任意 |  |
| 抜粋（一覧用） | `excerpt` | テキストエリア | 必須 | 一覧画面に表示される短い説明文。100文字程度 |
| 本文 | `body` | リッチエディタ | 必須 | 見出し、画像、リスト、リンク等を含められる本文 |

---

## 2. 採用情報（募集要項）※拡張検討
API名: `jobs`
APIの型: リスト形式
現状はハードコードですが、将来的に各拠点の募集要項をCMS化する場合のスキーマです。

| フィールド名 | フィールドID | 種類 | 必須 | 詳細設定・備考 |
| :--- | :--- | :--- | :--- | :--- |
| 職種名 | `title` | テキストフィールド | 必須 | 例: 「ルート営業」「加工業務」 |
| 拠点 | `location` | セレクトボックス | 必須 | 選択肢: 「築地本社」「大田営業部」「大阪営業部」「豊洲営業部」 |
| 雇用形態 | `employmentType`| セレクトボックス | 必須 | 選択肢: 「正社員」「契約社員」「アルバイト・パート」 |
| 仕事内容 | `description` | リッチエディタ | 必須 | |
| 応募資格 | `requirements`| リッチエディタ | 任意 | |
| 給与 | `salary` | テキストエリア | 必須 | |
| 勤務時間 | `hours` | テキストエリア | 必須 | |
| 休日・休暇 | `holidays` | テキストエリア | 必須 | |
| 待遇・福利厚生| `benefits` | テキストエリア | 任意 | |
| 募集状況 | `status` | セレクトボックス | 必須 | 選択肢: `open`（募集中）, `closed`（募集終了） |

---

## 3. 共通設定（サイト全体）※拡張検討
API名: `settings`
APIの型: オブジェクト形式

| フィールド名 | フィールドID | 種類 | 必須 | 詳細設定・備考 |
| :--- | :--- | :--- | :--- | :--- |
| サイトお知らせ | `globalNotice` | テキストフィールド | 任意 | 全ページのヘッダー上部に表示する緊急のお知らせ等 |
| 表示状態 | `noticeActive` | 真偽値（Boolean）| - | お知らせを表示するかどうか |
