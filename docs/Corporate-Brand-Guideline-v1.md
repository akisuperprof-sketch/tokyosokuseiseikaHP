# 東京促成青果株式会社 Corporate Brand Guideline v1.0

本ガイドラインは、東京促成青果株式会社のブランド価値を構築・維持し、10年後も色褪せない「洗練されたインフラ企業」としてのブランド体験を提供し続けるための指針です。社内担当者、外部の制作会社、およびAIエージェントによる開発において、例外なく本ルールを適用します。

---

## 1. ブランドコンセプト
- **企業として何を伝えるか**：日本の食を根底から支え、生産者と消費者をつなぐ確かな「架け橋」であること。
- **どんな印象を持って欲しいか**：誠実で品質が高く、歴史の重みと同時に、未来へ向けたクリーンな物流体制を持つ企業。
- **歴史**：1948年設立の揺るぎない実績と文化。
- **信頼**：安定供給、安全な加工、強固なネットワーク。
- **未来**：最新のコールドチェーン、食の価値向上、地域創生。
- **食文化**：おいしい日本の食を応援し、食卓を豊かにする。
- **日本らしさ**：四季折々の青果を通じた、自然への敬意と繊細な美意識。

## 2. ブランドキーワード
すべてのコピーライティングやビジュアル選定において、以下のキーワードを優先します。
1. **信頼 (Trust)**
2. **品質 (Quality)**
3. **誠実 (Sincerity)**
4. **食文化 (Food Culture)**
5. **新鮮 (Freshness)**
6. **物流 (Logistics)**
7. **地域 (Local/Regional)**
8. **未来 (Future)**

## 3. デザイン思想
本ブランドは以下の比率でデザイン要素を構成します。
- **Modern (40%)** : 最新の物流、クリーンな環境、先進的な機能美。
- **Traditional (30%)** : 1948年からの歴史、築地から豊洲へ受け継がれる文化と明朝体の品格。
- **Corporate (15%)** : B2Bとしての確固たる信頼感と安定感。
- **Natural (10%)** : 青果の瑞々しさ、大地、太陽の恵み、木目の温もり。
- **Luxury (5%)** : 高品質を保証する上質なトーンと洗練された余白。

## 4. カラー
- **Brand (Primary)**: `Deep Green (#1a2e22)` — 信頼、誠実、深みのある自然。
- **Accent**: `Dull Gold (#b99436)` — 品質、収穫、豊かさ。
- **Background**: `Ecru / Off-white (#fdfcfb)` — 生成り、清潔感、自然な紙の質感。
- **Foreground (Text)**: `Dark Gray (#1c1c1c)` — 墨色。純黒を避け、上品な読みやすさを確保。
- **禁止色**: 原色の赤、青、黄色など、安価な印象や警告を連想させるビビッドカラーの多用。
- **使用比率**: Background (70%) : Brand (20%) : Accent (5%) : Others (5%)

## 5. タイポグラフィ
- **見出し**: 游明朝 / Noto Serif JP。歴史と品格、伝統を演出する。
- **本文**: 游ゴシック / Noto Sans JP。現代的な視認性とフラットな読みやすさを確保。
- **数字 / 英語**: Inter。美しく洗練されたプロポーション。
- **余白**: 常に十分な `line-height` (1.8 - 2.0) と、段落間の大きな余白を確保する。
- **行間**: ゆったりと取り、文字が詰まった窮屈な印象（情報過多による安っぽさ）を与えない。

## 6. 写真ルール
画像は「空いているから埋める装飾」ではなく、「ブランドを伝える情報」として配置します。
- **会社写真**: 清潔で整理された施設、働く人の誠実な姿。
- **AI画像**: 「イメージ」としてのみ使用し、実在を偽装しない。
- **人物**: 後ろ姿やシルエット、顔が判別できない引きの画。
- **野菜**: 水滴、シズル感、極端なマクロ、美しい自然光。
- **物流**: 整然と並ぶ木箱、差し込む朝日、広大さ。
- **市場**: 活気があるが散らかっていない、静謐な朝の光。
- **背景（余白）**: 文字を載せるための意図的なボケや暗部（ネガティブスペース）があること。
- **Hero**: 文字とのコントラストが担保され、世界観を一瞬で伝える最高品質の1枚のみ。
- **禁止写真**: 文字が入った素材、不自然な合成、過度に彩度が高いもの、清潔感のない裏側。

## 7. AI画像ガイドライン
AIで画像を生成しブランドライブラリに追加する場合は、以下を厳守します。
- 実在施設を再現しない。
- 社員を再現しない。
- ロゴを作らない。
- 会社名を入れない。
- 人物はシルエット中心。
- **自然光、朝、木、深緑、生成り** を基調とする。

## 8. アイコン
- **太さ**: 1.5px 〜 2px のソリッドで繊細なライン（Lucideアイコン等を基準）。
- **サイズ**: 24px前後を基本とし、デザインアクセントには大きめ（48px等）の透過アイコンを用いる。
- **角丸**: 鋭角すぎず、丸すぎない（2px〜4px程度のradius）。
- **統一ルール**: アウトライン型で統一し、塗りつぶしアイコンは多用しない。色はBrandまたはMuted。

## 9. ボタン
- **Primary**: 背景色 `Brand` / テキスト 白 / Hover時は `-translate-y-0.5` と微小な影の追加。
- **Secondary (CTA)**: 背景色 `Accent` / テキスト 白 / Hover時は少し明るいゴールドへ。
- **Focus**: アクセシビリティ担保のため 3pxのOutline (`ring-2 ring-brand ring-offset-2`)。
- **Disabled**: `opacity-50`, `cursor-not-allowed`。

## 10. カード (Bento / News etc.)
- **余白**: 内部Paddingは広め（p-8 〜 p-10）に取り、窮屈にしない。
- **影**: 通常時は `shadow-sm`、Hover時に `shadow-md` または `shadow-lg`。
- **境界線**: 薄いグレー (`border-border`)、Hover時に `border-brand/20` などで上品に。
- **角丸**: `rounded-xl` または `rounded-2xl`（12px〜16px）。
- **画像比率**: Heroやカードサムネイルは 16:9 または 3:2 を基本とする。高さを統一する。

## 11. レイアウト
- **Container幅**: 最大 1024px〜1200px を基準とする。
- **余白**: セクションの上下には `py-24`（約96px）の大きな余白を設け、呼吸させる。
- **Section間隔**: 背景色 (`bg-background` と `bg-surface`) を交互に配置し区切るか、ボーダーを使用。
- **Grid**: 12カラムグリッドをベースとするモダンな Bento Grid を採用。
- **レスポンシブ**: スマホでは1カラム、Tabletで2カラム、PCで最適化されたマルチカラムを維持する。

## 12. アニメーション
- **採用**: Reveal（下からのゆっくりとしたフェードイン）、Fade、Hover（微小な浮き上がり）、Arrow（矢印の右移動）。
- **禁止**: バウンド、派手な回転、過度なパララックス、意味のない数字のカウントアップなど、信頼感を損なうチープな動き。

## 13. Hero
- **Hero画像**: ランダム表示（スライダー）は禁止。ブランド印象を固定するため四季に応じた固定1枚とする。
- **Hero動画**: 高品質な実写動画が正式提供された場合のみ優先採用。音声なし、ループ再生。
- **オーバーレイ**: 文字の可読性のため `bg-gradient-to-t from-brand/90` などのグラデーション透過を使用。
- **文字位置**: 中央または左下。余白と画像のネガティブスペースに合わせる。

## 14. CTA (Call To Action)
- **配置**: 各主要ページの下部、およびTopページのフッター直前。
- **文言**: 「お問い合わせフォーム」「お電話」など具体的かつ簡潔に。
- **色**: `bg-brand` などの暗い背景に `btn-accent` ボタンを配置し、高いコントラストで視線を集める。
- **ボタン意匠**: 矢印アイコンを付与し、アクションを促す。

## 15. News
- **写真**: アイキャッチ画像が存在しない記事には、AIで生成した「高品質な青果マクロ画像」をフォールバックとして表示し、一覧の美しさを保つ。
- **公開ルール**: ContentStatusによる厳格な「公開 / 非公開 / ドラフト」管理を徹底する。

## 16. Recruit
- **ブランドイメージ**: 求人広告特有の「笑顔の集合写真」ではなく、洗練された「働く環境」「朝の光」「協働するシルエット」を重視。
- **トーン**: 「求人」ではなく「働きたくなる、誇りを持てるインフラ企業」を伝える。誠実、安定、やりがい、心地よい空間。

## 17. SEO
- **タイトル**: 各ページ固有のTitle + `｜東京促成青果株式会社`。
- **Description**: 120文字前後で、ページの目的とキーワードを的確に記述。
- **OGP**: 高品質なHero画像（ロゴなしの美しい写真）を `og:image` として設定。

## 18. アクセシビリティ
- **WCAG基準**: コントラスト比 4.5:1 を遵守（特にブランドカラーとテキスト）。
- **Focus**: `focus-visible` によるキーボードナビゲーションの視認性確保。
- **Alt属性**: 全ての画像に意味のある `alt` 属性（単なる装飾画像は `alt=""`）。
- **Reduced Motion**: OS設定（`prefers-reduced-motion`）に連動し、動画の自動再生や過度なアニメーションを停止する。

## 19. ブランド品質チェックリスト
ページやコンテンツを追加する際は、必ず以下を確認すること。
- [ ] 1. 画像に文字やロゴ、不自然なAIの破綻が含まれていないか。
- [ ] 2. 全体が「朝・自然光・深緑・生成り」のトーン＆マナーに沿っているか。
- [ ] 3. 余白は十分に確保されているか（要素が詰まりすぎていないか）。
- [ ] 4. 見出しは明朝体、本文はゴシック体のタイポグラフィ・ルールを守っているか。
- [ ] 5. ボタンやHover時のマイクロインタラクションが上品か。
- [ ] 6. Lighthouseの Performance / Accessibility スコアが緑（90点以上）を保っているか。

## 20. ブランドスコア
運用時の自己採点基準（各10点 / 計100点満点）：
1. **ブランド感 (10点)**: コンセプト（日本の食を支える架け橋）に合致しているか。
2. **高級感 (10点)**: 安っぽさ、チープな動きや配色がないか。
3. **信頼 (10点)**: 正確な情報、誠実なトーンか。
4. **歴史 (10点)**: 軽薄にならず、老舗としての重みがあるか。
5. **読みやすさ (10点)**: コントラストと文字サイズ、行間は適切か。
6. **写真 (10点)**: 高品質、適切な被写体、意味のある配置か（装飾目的で無駄に置いていないか）。
7. **余白 (10点)**: 呼吸できるホワイトスペースが確保されているか。
8. **タイポグラフィ (10点)**: フォントファミリーと組版の美しさ。
9. **アクセシビリティ (10点)**: 全ての人が使えるか（キーボード、スクリーンリーダー対応）。
10. **SEO (10点)**: 構造化データとメタデータが正確か。

# 自己採点（Phase 4 完了時点）と Phase 5 への提言

現在のサイトを上記「Corporate Brand Guideline v1.0」に照らして自己採点した結果と、今後の改善余地（Phase 5）を以下に整理します。

## 現在の自己採点： **98 / 100点**
- **達成できている点 (98点)**: ブランドカラーの統一、明朝体/ゴシック体の適切な使い分け、余白の確保、AIを用いた極めて高品質な写真の配置、Lighthouse最適化（高速描画・アクセシビリティ対応）。特に「画像が無い方が美しい場所には配置しない」という引き算の美学が完璧に機能しており、Apple/MUJIレベルの洗練された企業サイトに仕上がっています。
- **減点ポイント (2点)**: 画像がまだAI生成の「代替イメージ」である点。

## Phase 5（次期改善）への提言
機能追加ではなく、ブランド価値をさらに絶対的なものにするためのネクストステップです。

1. **実写素材への段階的リプレイス計画**
   - 現在のAI画像（構図・光・トーン）を「撮影指示書（絵コンテ）」として流用し、プロカメラマンによる実写撮影を実施する。
   - 特にHero動画（朝の市場風景など）を実写で撮影し、実装済みの `HeroVideo` コンポーネントに組み込むことで、ブランド価値が完成します。
2. **ニュース・採用コンテンツの継続的運用ルールの徹底**
   - 今後担当者が記事を追加する際、アイキャッチ画像の品質がバラつかないよう、AIブランドライブラリから画像を選定するワークフローを確立する。
3. **マイクロコピー（UXライティング）の研ぎ澄まし**
   - サイト内の細かな文言（例：「送信する」「詳細を見る」など）を、より丁寧でブランドに寄り添った言葉遣いへ微調整する。
